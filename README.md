# jeremiaheth.github.io

Personal portfolio site for **Jeremiah Ojo** — Senior Cybersecurity Specialist, builder, and open-source contributor.

🔗 **Live:** [jeremiaheth.github.io](https://jeremiaheth.github.io)

## About

A dark-themed, performance-focused portfolio showcasing cybersecurity expertise, AI-driven security tooling, and hands-on projects. Built with modern web technologies and deployed via GitHub Pages.

## Tech Stack

- **Framework:** Next.js 14 (App Router, static export)
- **Styling:** Tailwind CSS
- **Animation:** GSAP + Framer Motion + Lenis smooth scroll
- **Icons:** Lucide React + custom SVGs
- **Deployment:** GitHub Actions → GitHub Pages

## Projects Featured

- **[neolata-mem](https://github.com/Jeremiaheth/neolata-mem)** — Graph-native memory engine for AI agents with Zettelkasten linking and biological decay
- **AI SOC Assistant** — Automated threat intelligence, vulnerability triage, and MITRE ATT&CK mapping
- **The Vanguard** — Multi-agent offensive security platform with four specialized AI agents

## Development

```bash
npm install
npm run dev        # http://localhost:3000
npm run build      # Static export to /out
```

## Deployment

Pushes to `main` trigger automatic deployment via GitHub Actions. The workflow builds the static export and deploys to GitHub Pages.
